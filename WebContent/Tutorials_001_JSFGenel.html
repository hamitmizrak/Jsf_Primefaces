JSF Genel Bilgiler

JSF bir framework yani bir çatıdır. Hazır bileşenler(komponentler/etiketler) JSP ile arasındaki fark AJAX kolayca entegre edebilmemizdir. Bir olay olur o olay üzerine işlem yapılır. JSF MVC standartıdır. Hibernate /Spring Frameworkler ile çalışabilmektedir. Java Programlama dilinin standart web çatısıdır. JSF 2.0 önce JSP kullanılıyordu. ancak JSF 2.0 'dan sonra Facelet mimarisine dönülmüştür. Bileşen tabanlı bir framewordktür.
